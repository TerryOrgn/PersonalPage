# TerryYuan 个人品牌页设计文档

**日期**: 2026-06-19
**目标**: 为 Terry Yuan 构建面向客户的解决方案架构师个人品牌页面
**参考**: https://mattwilldev.com/

---

## 1. 项目定位

- **目标用户**: 潜在甲方客户（金融/保险行业）
- **核心目标**: 展示方案能力与行业经验，促使用户联系洽谈合作
- **内容基调**: 专业、可信赖，强调行业深度与交付能力

## 2. 技术选型

| 项 | 选择 |
|---|---|
| 框架 | Next.js App Router |
| 样式 | Tailwind CSS |
| 内容管理 | 独立 TypeScript 数据文件 |
| 部署 | Vercel |
| 动效 | CSS transitions + framer-motion (轻量动画) |

## 4. 各模块详设

### 4.1 Navigation

- 固定顶部，毛玻璃半透明深色背景
- 左侧: Terry Yuan (品牌名)
- 右侧: 各区块导航链接，点击平滑滚动
- 滚动时添加底部细线分隔

### 4.2 Hero

- 深色 (#0a0a0a 或纯黑) 全宽背景
- 居中垂直排列:
  - 小号大写标签: "解决方案架构师"
  - 大标题: Terry Yuan (48-64px 粗体)
  - 副标题 (2-3行):
    - "15年企业级系统研发"
    - "专注金融投资 / 风控 / 保险等领域"
    - "客户：中国投资股份有限公司 · 太平洋保险公司 · 福特汽车金融公司"
  - 两个 CTA 按钮:
    - 主按钮: "查看案例" → 实心白/亮色填充 + 黑字
    - 次按钮: "联系我" → 白色描边 + 白字
- 按钮 hover: 微缩放 + 亮度变化

### 4.3 引言

- 居中引用块，较大字号，斜体或衬线字体以区分正文
- 内容: "以精准洞察需求为基，凭简单高效架构，借团队协作之力，权衡技术与成本，为您打造可持续演化的卓越解决方案"
- 左侧装饰性引号或竖线

### 4.4 About

- 居中段落，max-width 约 700px
- 3-4 句简介，大号正文
- 内容: 介绍自己是解决方案架构师，15年金融/保险行业经验，从需求分析到交付落地全流程能力，核心理念

### 4.5 Expertise

- 三列响应式卡片 (移动端堆叠)
- 每张卡片:
  - 小图标或编号
  - 标题: 企业级架构设计 / 行业解决方案 / 技术交付落地
  - 描述段落
  - 底部技术标签 (pill 样式)
- 卡片有微妙的边框和 hover 上浮效果

### 4.6 Cases

- 两列案例卡片 (移动端单列)
- 每张卡片:
  - 行业标签 (保险 / 汽车金融)
  - 客户名称
  - 项目标题
  - 简短描述 (1-2句)
  - 技术栈标签
  - "查看详情" 按钮 → 点击弹出 Modal
- Modal 详情弹窗:
  - 半屏居中弹窗，深色背景
  - 展示完整描述、技术架构要点、成果
  - 关闭按钮 + 点击遮罩关闭

### 4.7 Experience

- 居中时间线布局 (左侧时间 + 竖线 + 右侧内容)
- 三条经历，每条默认折叠显示概要 (公司 · 职位 · 时间 · 标签)
- 点击展开显示详细成果 bullet list
- 使用 CSS transition 展开动画

### 4.8 Contact

- 左右分栏 (移动端堆叠)
- 左栏表单:
  - 姓名、邮箱、主题、内容
  - 提交按钮
  - 提交后前端显示成功提示 (无后端)
- 右栏联系信息:
  - 邮箱
  - LinkedIn 链接
  - 微信 (可选)

### 4.9 Footer

- 居中布局
- 社交图标行: LinkedIn、GitHub、Email (SVG 图标)
- 底部文字: TERRY YUAN ©2026

## 5. 响应式设计

- 断点: sm (640px)、md (768px)、lg (1024px)
- 移动端: 单列布局，导航简化，字号缩小
- 平板: 两列布局，卡片适配
- 桌面: 完整三列/两列布局

## 6. 性能目标

- Lighthouse Performance ≥ 90
- 首屏加载 < 2s (3G)
- 图片优化: Next.js Image 组件 + WebP 格式
- 字体: next/font 加载系统字体优先

## 7. 不包含

- 后端 API / 邮件发送 (后续可接入)
- CMS 内容管理 (后续可升级)
- 博客 / 文章模块
- 多语言
- 暗/亮模式切换 (固定深色主题)
